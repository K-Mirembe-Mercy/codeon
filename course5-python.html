LESSON 01 · PYTHON BASICS

Hello, Africa! Your First Python Lines

Python reads like English and runs anywhere. In this lesson you'll write your first real script — not a toy example, but code that introduces you to the world in the way a Ugandan developer would.

🌍

Why Python is Africa's fastest-growing language

From M-Pesa's backend tools to Apollo Agriculture's data pipelines, Python powers much of Africa's most impactful technology. It is the language of data, automation, and AI — and Africa has more data problems to solve than any other region on earth.

PRINT
print("Hello, Uganda!")
Outputs text to the screen. Your first Python instruction.
VARIABLES
name = "Kafeero"
age = 20
Store data with a name. Strings use quotes, numbers don't.
INPUT
name = input("Your name: ")
Ask the user for information. Returns a string.
F-STRINGS
print(f"Hello {name}!")
Embed variables inside strings cleanly.
IF/ELSE
if age >= 18:
  print("Adult")
Run code only when a condition is true.
LOOP
for i in range(5):
  print(i)
Repeat code a set number of times.

LESSON 02 · DATA STRUCTURES

Lists, Dicts & Uganda's Markets

Data structures are how you organise information. In this lesson you'll build a market price database using Python lists and dictionaries — exactly the kind of tool Ugandan farmers need.

🌾

Real problem: market price information

Millions of Ugandan farmers sell produce without knowing current market prices. A simple Python dictionary of prices — updated regularly — can directly increase their income. This is what you're building today.

LIST
markets = ["Owino", "Nakasero", "Kalerwe"]
Ordered, changeable collection. Access by index [0].
DICT
price = {"maize": 1200, "beans": 3500}
Key-value pairs. Access by key: price["maize"].
LOOP DICT
for crop, ugx in prices.items():
  print(crop, ugx)
Iterate over all key-value pairs in a dictionary.
SORTED
sorted(prices, key=lambda x: x[1])
Sort a list or dict by value. Essential for rankings.

LESSON 03 · FUNCTIONS & MODULES

Reusable Code for Scalable Solutions

Functions let you write code once and use it everywhere. In this lesson you'll build a reusable health data module — the kind of tool health clinics across Uganda need.

🏥

Africa's health data challenge

Many African clinics record patient data on paper or in disconnected spreadsheets. A Python module that processes, validates, and reports on health data can transform a clinic's ability to identify trends and save lives.

FUNCTION
def greet(name):
  return f"Hello {name}"
def keyword defines a reusable block of code. return sends data back.
DEFAULT PARAM
def alert(msg, urgent=False):
Parameters can have default values — optional when calling.
LIST COMP
[x*2 for x in data if x > 0]
Build a new list with one clean line. Very Pythonic.
IMPORT
import statistics
statistics.mean(data)
Use Python's built-in modules for maths, dates, files, etc.

LESSON 04 · FILES & AUTOMATION

Automate the Boring Stuff in Uganda

File handling and automation are where Python earns money. Businesses pay well for scripts that process CSV reports, generate invoices, and automate repetitive tasks. You're building exactly that.

📁

Automation = Income

A Kampala accountant who manually processes 200 invoices per week can save 3 days of work with one Python script. That's a service worth UGX 500,000+ per month — and you can build it with what you learn today.

WRITE FILE
with open("report.txt","w") as f:
  f.write("data")
Create and write to files. Always use with-block.
READ FILE
with open("data.csv","r") as f:
  lines = f.readlines()
Read all lines from a file into a list.
CSV MODULE
import csv
reader = csv.DictReader(f)
Parse CSV files into dictionaries automatically.
OS MODULE
import os
os.makedirs("reports")
Create folders, list files, check paths exist.

LESSON 05 · APIs & WEB DATA

Connect Python to the World's Data

APIs let your Python scripts talk to the internet — fetch weather for Kampala farmers, get exchange rates for freelancers, retrieve news headlines, and much more. This is where Python becomes truly powerful.

🌐

APIs African developers use every day

MTN MoMo API for mobile payments. OpenWeather for agricultural forecasts. Exchange Rate APIs for freelancers pricing in USD. Twilio for WhatsApp alerts. Every major African tech product uses APIs — and now you will too.

REQUESTS
import requests
r = requests.get(url)
data = r.json()
The most important Python library for web data. Install: pip install requests
JSON
import json
json.dumps(data)
json.loads(text)
Convert between Python dicts and JSON strings.
STATUS CODE
if r.status_code == 200:
  print("Success")
200 = OK. 404 = Not Found. 500 = Server Error. Always check.
ENV VARS
import os
key = os.getenv("API_KEY")
Never hardcode API keys. Store in environment variables.
